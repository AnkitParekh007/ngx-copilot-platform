# What This Proves

## Technical Proof

This repo shows how to structure an Angular copilot SDK around stable contracts instead of hard-coding one app-specific chat experience. It includes configuration, context serialization, RAG normalization, streaming, tool registry behavior, approvals, and a reusable shell surface.
