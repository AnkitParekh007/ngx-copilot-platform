# Integration Guide

1. Install the package or copy the SDK pattern into your workspace.
2. Configure `provideCopilot` in app providers.
3. Add `ngx-copilot-shell` to a target route.
4. Provide safe `CopilotContext` from route, role, tenant, and selected record.
5. Register tools through a backend-governed registry.
6. Render RAG sources and approval cards for transparency.
7. Keep model/provider secrets server-side.
