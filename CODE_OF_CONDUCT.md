# Code of Conduct

This project uses a simple professional standard: be respectful, specific, and constructive.

## Expected Behavior

- Keep discussions technical and useful.
- Assume good intent.
- Give actionable feedback.
- Respect different experience levels.

## Unacceptable Behavior

- Harassment or personal attacks.
- Spam or promotional noise.
- Sharing secrets, private data, or proprietary code.

Maintainers may close issues or PRs that do not follow these expectations.
